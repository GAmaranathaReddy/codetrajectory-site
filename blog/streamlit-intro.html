Getting Started with Streamlit

Streamlit is an open-source Python library that makes it easy to create and share beautiful, custom web apps for machine learning and data science. In just a few minutes, you can build and deploy powerful data apps.

Why Streamlit?

  • Super simple to use — turn Python scripts into web apps with minimal code.
  • Great for rapid prototyping and sharing results.
  • Supports interactive widgets, charts, and media.

Installation

pip install streamlit

Your First App

import streamlit as st

st.title("Hello, Streamlit!")
st.write("Welcome to your first Streamlit app.")

Save this as app.py and run:

streamlit run app.py

Key Features

  • Widgets: st.button, st.slider, st.selectbox, etc.
  • Charts: st.line_chart, st.bar_chart, st.map
  • Media: st.image, st.audio, st.video
  • Layout: Columns, expanders, sidebar, etc.
